# 📝 Sistema de Cadastro de Editais de Monitoria

Programa desenvolvido para automatizar e facilitar a criação de editores para a seleção de novos monitores acadêmicos.

## 🚀 Funcionalidades
* **Cadastro de Editais:** Registro de disciplinas, vagas e prazos.
* **Listagem:** Visualização de todos os artigos abertos e encerrados.
* **Validação:**Verificação automática de dados e requisitos mínimos.
* **Persistência: Armazenamento dos dados em[Xstream e Java]

## 🛠️ Tecnologias
* **Linguagem: Java 
* **IDE:** Eclipse
